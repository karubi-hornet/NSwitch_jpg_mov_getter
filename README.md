# NSwitch_jpg_mov_getter
NSwitchの画像と動画をそれぞれ1フォルダに集約

## Usage
1. NSwitchの"Album"フォルダをこのフォルダにコピー
2. NSwitch_igp_mov_getter.shの実行権限を変更
    ```shell
    chmod +x NSwitch_igp_mov_getter.sh
    ```
3. 実行
    ```shell
    ./NSwitch_igp_mov_getter.sh
    ```
4. `img`フォルダには*.jpgファイルが、`mov`フォルダには*.mp4ファイルが移動される。
